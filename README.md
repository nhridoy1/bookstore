# 📚 Online Book Booking System

## 📌 Overview
The **Online Book Booking System** is a web-based application that allows users to browse, buy, and borrow books online. It also provides an admin panel to manage books, users, and transactions.

This system aims to simplify book purchasing and borrowing through an easy-to-use digital platform.

---

## 🎯 Project Purpose
The main purpose of this project is to:
- Provide a centralized platform for book buying and borrowing
- Enable online payments and cash on delivery options
- Help admins efficiently manage books and users

---

## 👥 Target Audience
- Students
- Book readers
- Libraries
- General users interested in online book services

---

## 🚀 Features

### 👤 User Features
- User Registration & Login
- Browse available books
- Search books by title or category
- Buy books online
- Borrow books
- Payment options:
  - Online Payment
  - Cash on Delivery
- View order history

---

### 🛠️ Admin Features
- Admin Login Dashboard
- Add new books
- Update book details
- Delete books
- Manage users
- Monitor orders and transactions

---
